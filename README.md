# AIDOC
BRIDGING HEALTHCARE GAPS IN RURAL INDIA
🏥 Bridging Healthcare Gaps in Rural India with AI

An AI-powered multilingual healthcare assistant designed to support rural communities by providing instant health guidance through voice and text input in local languages.

📌 Problem Statement

65% of India’s population lives in rural areas.

Severe shortage of doctors in villages.

Lack of awareness, language barriers, and distance from health centers delay treatment.

Result → Late diagnosis and preventable deaths.

💡 Solution

Our system provides:

🌐 Multilingual AI Health Chatbot (Voice + Text support)

🩺 Symptom-based illness suggestions (guidance, not final diagnosis)

🏥 Nearest government health center locator

🧾 Basic first-aid and preventive care advice

⚙️ System Workflow

User inputs symptoms (Voice/Text)

Speech-to-Text conversion (regional languages supported)

NLP + ML model processes symptoms

AI suggests possible illness & treatment guidance

Location-based API finds nearest health center
